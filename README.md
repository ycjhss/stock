# 주식 인텔리전스 — AI 투자분석 플랫폼

> AI 기반 한국 주식 분석 · 숨겨진 유망주 발굴 · 실시간 글로벌 연동

## 🔗 사이트 주소
배포 후 이 곳에 주소가 생성됩니다:  
`https://[내 GitHub 아이디].github.io/stock-intelligence/`

## 📌 주요 기능
- **종목 정밀분석** — 10대 요인 AI 분석 + 레이더 차트 + 기간별 전망
- **숨겨진 유망주 발굴** — 시장 소외 저평가 유망 종목 AI 발굴
- **시장·글로벌 분석** — 코스피·코스닥·글로벌 실시간 동향
- **포트폴리오 진단** — 보유 종목 리밸런싱 및 최적화 전략

## ⚙️ 사용 방법
1. 사이트 접속 후 **설정** 탭에서 Anthropic API 키 입력
2. API 키는 [console.anthropic.com](https://console.anthropic.com) 에서 무료 발급
3. API 키는 본인 브라우저에만 저장 (서버 전송 없음)

## ⚠️ 면책 조항
이 플랫폼은 투자 정보 제공 목적이며, 실제 투자 결정은 본인의 판단과 책임 하에 진행하시기 바랍니다.
